# 第一部分：项目基本信息

1. 项目名称（输入框）
    
2. 预期起止时间（日期范围选择器）
    
3. 所属专项（下拉列表）：一级学科
    
4. 所属领域（下拉列表）：科研相关的研究领域
    
5. 项目人数（数字输入框）
    

  

# 第二部分：项目组主要成员信息（修改版本）

成员信息使用列表式，表单填写人可以增减成员

### 项目负责人信息：

1. 姓名（输入框）
    
2. ORCID（选填，输入框）
    
3. 联系邮箱（输入框）
    
4. 职称（下拉列表）
    
5. 学历（下拉列表）
    
6. 简介（选填，输入框、限制200字符）：请简要描述项目主要负责人的主要研究方向、科研/项目成果（如代表性论文/专利）、项目经验（如已获得基金、项目等）等及在项目中承担的任务
    

  

### 成员信息：

1. 角色（下拉列表）：主要成员、其他成员
    
2. 成员简历（文档上传，限一份）：简历说明：描述曾参与的相关项目经验或背景，及简要描述在项目中承担的任务

# 第三部分：项目介绍

1. 项目简介（长文本输入框，限1500字内）：介绍目前存在的科学问题，可以通过什么手段解决此问题。
    
    1. 备注：支持上传文件读取内容填入【AI】
        
2. 实施的背景和意义（长文本输入框，限1500字内）：概述项目所面向的经济、社会和科技发展等有效需求，项目的先进性、重要性、必要性、可行性以及在行业发展中的地位和作用；预期实现的经济和社会效益等方面。
    
    1. 备注：支持上传文件读取内容填入【AI】
        
3. 项目里程碑（列表式模版）：每个项目一共三个阶段：
    
    1. 阶段（固定三行）：初期/中期/末期
        
    2. 起止时间（日期范围选择器）：每个阶段的起止时间
        
    3. 主要研究内容（输入框）：根据上述的初期/中期/末期时间，简述每个阶段的研究内容
        
    4. 预期目标（输入框）：详述该阶段研究的目标，目标应该符合SMART原则（比如项目预计产生论文数/发明专利/实用新型专利/PCT），每个阶段可存在多个目标

信息说明：

1. 经费类别（下拉选择）：
    
    1. 设备费：包括设备购置、设备租赁、设备改造等费用
        
    2. 业务费：包括耗材、加工、服务费、差旅费、会议费等相关费用
        
    3. 人力费：人力资源包括人员、劳务、专家咨询费用
        
    4. 管理费：包括场地管理比如水电气暖、绩效支出等
